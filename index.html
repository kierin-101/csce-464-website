Projects I've Worked On

Pitivi

Pitivi is a free and open-source video editor for Linux. It is written in Python and uses the GStreamer multimedia framework. My team "contributed" (the changes were never submitted in a merge request) a new feature that allowed the user to essentially bookmark points in their video timeline, give the bookmakrs names, do things like that.

(It was kinda boring, but it was a good learning experience. I learned a lot about the Git workflow and how to work with a team of developers. I learned a lot about the GStreamer framework and how to work with it. I also also learned that I don't like working with GStreamer.)

Agricultural Input Origination Software

(Sorry for the ugly white background on the logo. I couldn't find a good image of the logo with a transparent background.)

The Agricultural Input Origination Software (long name, blame the client) is a web application that allows users from Henderson State Bank and their customers to edit, view, and create loans. I mainly work on the front-end of the application, using ReactJS and TypeScript to create a User Interface for the application. (The reason I work on the front-end is because I'm scared of the back-end. I don't like working with databases.)

I would say more about working with Henderson, but I have no idea what I'm allowed to say. I don't want to get in trouble for accidentally leaking sensitive information, so I'll just leave it at that.
